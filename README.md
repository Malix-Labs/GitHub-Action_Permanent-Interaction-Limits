# GitHub-Action_Limit-Interactions

## Features

- Cyclically re-enable interaction limits ([repository](https://github.com/<USERNAME>/<REPOSITORY>/settings/interaction_limits), [organization](https://github.com/organizations/<ORGANIZATION>/settings/interaction_limits), and [user](https://github.com/settings/interaction_limits) ones) you have set for 6 months (which is the maximum)
- On opened issues by allowed users (Up to Existing Users, Contributors, Collaborators, _(for Organizations:)_ Organization members): lock
- On opened issues by forbidden users (Up to New Users, Existing Users, Contributors, _(for Organizations:)_ Collaborators): lock, delete, close as not planned, move, or convert to discussion
